# Multi-Method Secure Password Generator

Набор инструментов на Python для создания криптографически стойких паролей и токенов.

## Функционал
Скрипт включает два метода генерации:

1. Custom Secure Password (`generate_secure_password`):
   Полный набор символов (буквы, цифры, пунктуация).
   Валидация сложности: Скрипт гарантирует наличие всех типов символов (минимум 1 заглавная, 1 строчная, 1 цифра и 1 спецсимвол).
   
2. URL-Safe Token (`generate_token_password`):
   Использует нативный метод `secrets.token_urlsafe`.
   Идеально подходит для временных паролей, токенов сессий или API-ключей, которые нужно передавать в ссылках.

## Безопасность
Используется модуль `secrets` (библиотека стандартных компонентов Python), который задействует системные источники энтропии ОС, подходящие для криптографических задач.

## Использование
Запустите скрипт через терминал: bash python password_generator.py
