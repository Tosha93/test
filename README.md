# Multi-Method Secure Password Generator

Набор инструментов на Python для создания криптографически стойких паролей и токенов.

## Функционал
Скрипт включает два метода генерации:

1. Custom Secure Password (`generate_secure_password`):
   Полный набор символов (буквы, цифры, пунктуация).
   Валидация сложности: Скрипт гарантирует наличие всех типов символов (минимум 1 заглавная, 1 строчная, 1 цифра и 1 спецсимвол).
   
2. URL-Safe Token (`generate_token_password`):
   Использует нативный метод `secrets.token_urlsafe`.
   Идеально подходит для временных паролей, токенов сессий или API-ключей, которые нужно передавать в ссылках.

## Безопасность
Используется модуль `secrets` (библиотека стандартных компонентов Python), который задействует системные источники энтропии ОС, подходящие для криптографических задач.

## Использование
Запустите скрипт через терминал: bash python password_generator.py

# Multi-Method Secure Password Generator

Набор инструментов на Python для создания криптографически стойких паролей и токенов с анализом надежности.

## Функционал

Скрипт включает три основных метода:

 1. Custom Secure Password (`generate_secure_password`)
- **Полный набор символов**: буквы (прописные и строчные), цифры, пунктуация
- **Валидация сложности**: Гарантирует наличие всех типов символов
  - Минимум 1 заглавная буква
  - Минимум 1 строчная буква
  - Минимум 1 цифра
  - Минимум 1 спецсимвол
- Параметры: Полностью настраиваемая длина и минимальные количества символов каждого типа
- Перемешивание: Криптографически безопасное перемешивание с использованием `secrets.randbelow()`

2. URL-Safe Token (`generate_token_password`)
- Метод: Использует `secrets.token_urlsafe` (base64-кодирование)
- Безопасность: URL-безопасные символы (буквы, цифры, дефис, подчеркивание)
- Применение: Идеально для:
  - Временных паролей
  - Токенов сессий
  - API-ключей
  - Ссылок и URL-параметров

3. Анализ Надежности (`check_password_strength`)
- Проверка типов символов: Наличие букв, цифр, спецсимволов
- Оценка длины: Особые баллы за длину ≥12 и ≥20 символов
- Уровни надежности: От "Слабый" до "Очень сильный"
- Подробный отчет: Возвращает словарь со всеми параметрами

### Параметры командной строки:

```bash
# Только надежный пароль длиной 25 символов
python password_generator.py --type secure --length 25

# Только URL-safe токен
python password_generator.py --type token --length 32

# С анализом надежности
python password_generator.py --check-strength

# Оба типа паролей с проверкой надежности
python password_generator.py --type both --length 30 --check-strength
```

## Примеры вывода
